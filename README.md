**Setup**

1) Clone this repository into htdocs
2) Move config.php into the repository's parent folder
3) Launch XAMPP
4) Import ConcertSetlist.sql in phpMyAdmin

**Use**

CREATE a new song entry by entering the song's name, album, and artist and then pressing "Add Song".
READ the setlist by looking at the screen.
UPDATE the setlist's order by using the up and down arrows.
DELETE a song by using the trash can icon.
